Mengenal Fungsi Google Docs

Belajar mengolah kata secara cerdas di era Cloud.

1. Apa itu Google Docs?

Google Docs adalah aplikasi pengolah kata berbasis Cloud (Awan). Berbeda dengan aplikasi tradisional, Google Docs memungkinkan pengguna untuk bekerja secara kolaboratif (bersama-sama) dalam satu file secara real-time.

Keunggulan Utama:
  • Auto-save: Tersimpan otomatis setiap kali Anda mengetik.
  • History Versi: Melihat riwayat pengeditan dan siapa yang mengubahnya.
  • Akses Lintas Perangkat: Buka di HP, Tablet, atau PC kapan saja.

2. Format File Penting

Ekstensi Nama Format Kegunaan
.docx Microsoft Word Untuk kompatibilitas dengan aplikasi offline.
.pdf Portable Document Untuk dokumen final agar tata letak tidak berubah.
